Device Classifier

Device Classifier Destinguishes between Smartphone, Tablets, Laptops and PCs

This is a short example Projekt provided with Starlet and a basic image classifier made with the Fast.ai Framework. This project can destinguish different pictures based on the trained Model in the background. It was trained to recognize pictures which include the following:

  • Gaming Laptop
  • Gaming PC
  • Smartphone
  • Tablet

How to use this projekt

Steps:

  1. Upload picture on the left side
  2. Click Analyse
  3. You should now see the result of the classifier in the Chart
